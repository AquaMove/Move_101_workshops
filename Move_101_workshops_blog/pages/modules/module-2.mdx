# Module 2: Basic Types and Functions

In this module, you'll explore Move's basic data types, functions, and control flow through a calculator application. You'll learn how to work with primitive types, implement functions with different visibility levels, and handle errors.

## üéØ Learning Objectives

By the end of this module, you will be able to:

- Master Move primitive types (u64, vector, etc.)
- Understand function signatures and visibility
- Implement error handling with `abort`
- Work with mutable and immutable references
- Use vector operations effectively

## üìö Key Concepts

### 1. Basic Types

Move provides several primitive types:

```move
u64: unsigned 64-bit integer
vector<T>: dynamic array
String: UTF-8 string
bool: boolean value
address: 32-byte address
```

### 2. Function Types

Move functions have different visibility levels:

```move
public entry fun: Callable from transactions
public fun: Callable from other modules  
fun: Private function (module-only)
```

### 3. Error Handling

Move uses `abort` for error handling:

```move
const EInvalidOperation: u64 = 0;

if (divisor == 0) {
    abort EInvalidOperation
};
```

## üõ†Ô∏è Step-by-Step Implementation

### Step 1: Create the Calculator Module

Create `sources/basic_types.move`:

```move
module basic_types::basic_types {
    use std::string::{Self, String};
    use sui::object::{Self, UID};
    use sui::transfer;
    use sui::tx_context::{Self, TxContext};

    /// A simple calculator object that demonstrates basic types and functions
    public struct Calculator has key, store {
        id: UID,
        value: u64,
        history: vector<u64>
    }

    /// Error codes
    const EInvalidOperation: u64 = 0;
    const EOverflow: u64 = 1;

    /// Create a new calculator with initial value
    public entry fun create_calculator(initial_value: u64, ctx: &mut TxContext) {
        let calculator = Calculator {
            id: object::new(ctx),
            value: initial_value,
            history: vector::empty()
        };
        transfer::public_transfer(calculator, tx_context::sender(ctx));
    }

    /// Add a number to the current value
    public entry fun add(calculator: &mut Calculator, amount: u64) {
        let new_value = calculator.value + amount;
        vector::push_back(&mut calculator.history, calculator.value);
        calculator.value = new_value;
    }

    /// Subtract a number from the current value
    public entry fun subtract(calculator: &mut Calculator, amount: u64) {
        if (calculator.value < amount) {
            abort EInvalidOperation
        };
        let new_value = calculator.value - amount;
        vector::push_back(&mut calculator.history, calculator.value);
        calculator.value = new_value;
    }

    /// Multiply the current value by a number
    public entry fun multiply(calculator: &mut Calculator, factor: u64) {
        let new_value = calculator.value * factor;
        vector::push_back(&mut calculator.history, calculator.value);
        calculator.value = new_value;
    }

    /// Divide the current value by a number
    public entry fun divide(calculator: &mut Calculator, divisor: u64) {
        if (divisor == 0) {
            abort EInvalidOperation
        };
        let new_value = calculator.value / divisor;
        vector::push_back(&mut calculator.history, calculator.value);
        calculator.value = new_value;
    }

    /// Get the current value
    public fun get_value(calculator: &Calculator): u64 {
        calculator.value
    }

    /// Get the calculation history
    public fun get_history(calculator: &Calculator): vector<u64> {
        calculator.history
    }
}
```

## üìñ Code Explanation

### Struct Definition
```move
public struct Calculator has key, store {
    id: UID,
    value: u64,
    history: vector<u64>
}
```

- `value`: Stores the current calculator value
- `history`: Vector storing previous values

### Error Constants
```move
const EInvalidOperation: u64 = 0;
const EOverflow: u64 = 1;
```

- Define error codes for different failure scenarios

### Function Parameters
```move
public entry fun add(calculator: &mut Calculator, amount: u64)
```

- `&mut Calculator`: Mutable reference to calculator object
- `amount: u64`: Immutable parameter

### Vector Operations
```move
vector::push_back(&mut calculator.history, calculator.value);
```

- `vector::empty()`: Create empty vector
- `vector::push_back()`: Add element to end
- `vector::length()`: Get vector size
